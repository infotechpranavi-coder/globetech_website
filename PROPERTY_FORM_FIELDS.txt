SUMMARY

TOTAL INPUT FIELDS: 40+ (including dynamic array inputs)

REQUIRED FIELDS (*):
- Property Name
- Location
- Developer
- Description
- At least one Image

OPTIONAL FIELDS:

DYNAMIC ARRAYS (Can add multiple entries):
- Highlights (Section 4)
- Pricing & Floor Plans (Section 6)
- Amenities (Section 7)
- Facilities (Section 8)

- Videos (Section 13, if exists)

FORM ACTIONS
================================================================================
- Submit Button: Saves property to database
- Cancel/Close Button: Closes modal without saving
- Remove Buttons: Available for all dynamic array entries

================================================================================
VALIDATION RULES
================================================================================
1. Property Name: Required, cannot be empty
2. Description: Required, cannot be empty
3. Developer: Required, must select from dropdown
4. Location: Required, cannot be empty
5. Images: At least one image required (either uploaded file or URL)
6. Price: Either base price or at least one pricing entry required
7. File Upload: Max 10MB, only image formats allowed
